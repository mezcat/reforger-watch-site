Reforger Watch

Live server tracking for Arma Reforger • Real-time players • Mod lists • Discord integration
➕ Invite Bot to Discord

🎮 Live Player Count

Displays current players / max players with a clean progress bar. Updates every 60 seconds.

📦 Mod List Viewer

Click the "Mods" button to see all server mods with versions & IDs. Search mods by name.

🔧 Easy Setup

Admin-only /setup command – add, edit, or remove servers. Automatically creates a dedicated category and channel.

🌍 Region & Info

Shows scenario, BattlEye status, password protection, official server flag, and optional IP display.

🔄 Auto‑rename

Channel name updates with player count (e.g., 🟢 EU1 • 42 Players Online). Handles rate limits gracefully.

📊 Global Tracking

Supports up to 8 servers per Discord guild. All data stored in a fast MySQL database.
